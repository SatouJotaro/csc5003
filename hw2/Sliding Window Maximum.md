### 问题分析

这道题目要求计算滑动窗口最大值。给定一个长度为 `n` 的整数数组 `nums` 和窗口大小 `k`，我们需要在数组上滑动一个大小为 `k` 的窗口，从左到右移动窗口，并输出每个窗口内的最大值。

- **输入**：第一行包含两个整数 `n` 和 `k`，第二行包含 `n` 个整数 `nums[i]`。
- **输出**：第一行输出窗口数量 `m = n - k + 1`，第二行输出 `m` 个整数，每个是相应窗口的最大值，用空格分隔。
- **约束**：
  - `1 ≤ n ≤ 10^5`
  - `-10^4 ≤ nums[i] ≤ 10^4`
  - `1 ≤ k ≤ n`
- **知识点**：滑动窗口最大值问题，通常使用单调队列（Monotonic Queue）解决，时间复杂度为 O(n)，空间复杂度为 O(k)。
- **解题思路**：使用一个双端队列（deque）来存储数组元素的索引，保持队列中的值按递减顺序排列。队列的前端始终是当前窗口的最大值。具体步骤：
  1. 遍历数组，对于每个元素：
     - 维护队列递减：如果队列尾部的值小于当前元素值，则弹出尾部索引，直到队列为空或尾部值大于等于当前值。
     - 将当前元素索引加入队列尾部。
     - 如果队列头部索引不在当前窗口内（即索引小于当前索引 `i - k + 1`），则弹出头部索引。
     - 当当前索引 `i >= k - 1` 时（即窗口已形成），将队列头部索引对应的值加入结果。
  2. 输出结果。

### C++ 代码实现

```cpp
#include <iostream>
#include <deque>
#include <vector>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    vector<int> nums(n);
    for (int i = 0; i < n; i++) {
        cin >> nums[i];
    }
    
    deque<int> dq; // 存储索引，值按递减顺序
    vector<int> result;
    
    for (int i = 0; i < n; i++) {
        // 维护队列递减：移除队列尾部小于当前值的索引
        while (!dq.empty() && nums[dq.back()] <= nums[i]) {
            dq.pop_back();
        }
        // 添加当前索引到队列尾部
        dq.push_back(i);
        
        // 移除不在当前窗口内的索引（头部索引小于 i - k + 1）
        if (!dq.empty() && dq.front() <= i - k) {
            dq.pop_front();
        }
        
        // 当窗口形成时（i >= k - 1），将当前窗口最大值加入结果
        if (i >= k - 1) {
            result.push_back(nums[dq.front()]);
        }
    }
    
    int m = n - k + 1;
    cout << m << endl;
    for (int i = 0; i < m; i++) {
        cout << result[i];
        if (i < m - 1) cout << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 代码解释

1. **输入读取**：读取 `n` 和 `k`，然后读取数组 `nums`。
2. **单调队列初始化**：使用 `deque<int>` 存储索引，`vector<int>` 存储结果。
3. **遍历数组**：
   - **维护队列递减**：当队列非空且队列尾部的值小于当前元素值时，弹出尾部索引，确保队列中的值递减。
   - **添加当前索引**：将当前索引 `i` 加入队列尾部。
   - **移除过期索引**：如果队列头部索引小于 `i - k + 1`，表示不在当前窗口内，弹出头部索引。
   - **记录窗口最大值**：当 `i >= k - 1` 时，队列头部索引对应的值是当前窗口最大值，加入结果。
4. **输出结果**：先输出窗口数量 `m`，然后输出结果向量中的最大值，用空格分隔。

### 示例验证

对于示例输入：
```
8 3
1 3 -1 -3 5 3 6 7
```
- 窗口数量 `m = 8 - 3 + 1 = 6`。
- 滑动窗口最大值为 `[3, 3, 5, 5, 6, 7]`。

代码输出：
```
6
3 3 5 5 6 7
```

### 关键点
- 单调队列确保每个元素最多被加入和移除一次，时间复杂度 O(n)。
- 队列存储索引而非值，方便判断元素是否在窗口内。
- 处理边界条件（如 `k=1` 或 `k=n`）时，代码自动适应。
