这道题是字符串算法中最经典的 **KMP (Knuth-Morris-Pratt) 算法**题目。

---

### 1. 题目解读

*   **输入**：
    *   第一行：字符串 $S$（长度为 $n$）。
    *   第二行：字符串 $P$（长度为 $m$）。
*   **输出**：
    *   第一部分：打印出 $P$ 在 $S$ 中出现的所有起始位置（从 1 开始计数，按从小到大顺序）。
    *   第二部分：打印出 $P$ 的 `pi` 数组（即前缀函数数组），也就是对每一个 $i$ (1 到 $m$)，求 $P[1..i]$ 的最长公共前后缀长度。
*   **约束**：
    *   $n \le 2 \times 10^6$，$m \le 2 \times 10^5$。这意味着我们需要 $O(n+m)$ 的线性时间复杂度算法，普通的嵌套循环 $O(n \times m)$ 会超时。

---

### 2. 核心知识点：KMP 算法与前缀函数

#### 什么是 `pi` 数组（前缀函数）？
`pi[i]` 的定义是：字符串前 $i$ 个字符组成的子串中，**最长的真前缀和真后缀相等的长度**。
*   “真前缀”：不包含字符串本身的开头部分。
*   “真后缀”：不包含字符串本身的结尾部分。

**例子：** $P = \text{"ABAB"}$
*   $i=1, P[1]=\text{'A'}$: 无真前缀/后缀，`pi[1]=0`
*   $i=2, P[1..2]=\text{"AB"}$: 无匹配，`pi[2]=0`
*   $i=3, P[1..3]=\text{"ABA"}$: 前缀'A'==后缀'A'，`pi[3]=1`
*   $i=4, P[1..4]=\text{"ABAB"}$: 前缀"AB"==后缀"AB"，`pi[4]=2`
*   结果：`0 0 1 2`

---

### 3. 解题思路

#### 第一步：计算 `pi` 数组
我们利用动态规划的思想。假设已知 `pi[1...i-1]`，在计算 `pi[i]` 时：
1. 令 `j = pi[i-1]`。
2. 不断检查 `P[j+1]` 是否等于 `P[i]`。
3. 如果相等，则 `pi[i] = j + 1`；如果不相等，则通过 `j = pi[j]` 不断回退，直到匹配或 `j` 归零。

#### 第二步：匹配过程
与计算 `pi` 数组类似。在遍历 $S$ 时，维护一个指针 `j` 指向当前匹配到 $P$ 的位置：
1. 如果 $S[i] == P[j+1]$，则 `j` 前进一位。
2. 如果 $j$ 等于 $m$，说明匹配成功，记录位置，并将 `j` 更新为 `pi[j]`（继续寻找下一个可能的重叠匹配）。

---

### 4. C++ 实现

```cpp
#include <iostream>
#include <vector>
#include <string>

using namespace std;

int main() {
    // 提升 I/O 速度
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string S, P;
    cin >> S >> P;

    int n = S.length();
    int m = P.length();

    // 1. 计算 pi 数组
    // 为了方便，这里采用 1-based indexing 逻辑的数组大小
    vector<int> pi(m + 1, 0);
    for (int i = 2, j = 0; i <= m; i++) {
        while (j > 0 && P[j] != P[i - 1]) j = pi[j];
        if (P[j] == P[i - 1]) j++;
        pi[i] = j;
    }

    // 2. 匹配过程
    for (int i = 0, j = 0; i < n; i++) {
        while (j > 0 && P[j] != S[i]) j = pi[j];
        if (P[j] == S[i]) j++;
        if (j == m) {
            // 输出 1-based 起始位置
            cout << i - m + 2 << "\n";
            j = pi[j];
        }
    }

    // 3. 输出 pi 数组
    for (int i = 1; i <= m; i++) {
        cout << pi[i] << (i == m ? "" : " ");
    }
    cout << endl;

    return 0;
}
```

### 代码要点总结：
1.  **下标处理**：虽然 C++ 字符串是 0-indexed，但 `pi` 数组定义通常是 1-based，代码中通过 `P[i-1]` 巧妙地衔接了这两者。
2.  **指针回退**：`while (j > 0 && P[j] != S[i]) j = pi[j];` 是 KMP 的灵魂，它确保了我们不需要回溯 $S$ 的搜索指针，保证了 $O(n)$ 时间复杂度。
3.  **效率**：使用了 `ios::sync_with_stdio(false)`，这在处理 $2 \times 10^6$ 数据量时非常重要，防止输入输出成为瓶颈。
